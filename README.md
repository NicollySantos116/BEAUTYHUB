# 💄 BeautyHub

Aplicativo mobile de catálogo de maquiagem desenvolvido com React Native (Expo).

## 📱 Sobre :
O BeautyHub permite visualizar produtos, acessar detalhes e simular compras de forma simples.

## 🚀 Tecnologias :
- React Native
- Expo
- JavaScript
- FlatList

## 🧩 Funcionalidades :
- Listagem de produtos
- Tela de detalhes
- Botões de interação (comprar/adicionar)
- Navegação entre telas
